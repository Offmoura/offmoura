## 🔥 Projeto em Destaque

### [Broken Friday](https://github.com/Offmoura/broken-friday) *(em construção)*
> Marketplace open-source **propositalmente quebrado** para ensinar DevOps/SRE na prática.

Um e-commerce fictício construído com falhas reais injetadas — race conditions, connection pool esgotado, deploy sem health check, observabilidade zero. O dev clona, observa o caos e aprende a consertar.

**Stack:** Docker Compose · PostgreSQL · Redis · RabbitMQ · Prometheus · Grafana

**Labs disponíveis:**
- 🔵 **SRE Lab 1** — O Banco Travou (Query N+1, connection pooling)
- 🔵 **SRE Lab 2** — O Estoque Vendeu Mais do que Tinha (Race Condition, ACID)
- 🔵 **SRE Lab 3** — Ninguém Sabia que Estava Falhando (Golden Signals, SLOs)
- 🟠 **DevOps Lab 1** — A API Não Aguentou (Horizontal Scaling, Rate Limiting)
- 🟠 **DevOps Lab 2** — Tudo Ficou Lento (Cache Redis, CDN)
- 🟠 **DevOps Lab 3** — O Deploy Quebrou Tudo (Rolling Update, Rollback)
